# 🌐 Dr. Divya Mishra — Portfolio Website

## Files
- `index.html` — Main portfolio page
- `style.css`  — Complete stylesheet
- `script.js`  — Dark mode, animations, navigation

## Deploy to GitHub Pages (5 minutes)

### Step 1 — Create a new GitHub repository
- Go to github.com → Click "+" → "New Repository"
- Name it EXACTLY: `divas-coder.github.io`
- Set to **Public**
- Do NOT add a README
- Click "Create repository"

### Step 2 — Upload the files
- Click "uploading an existing file"
- Drag `index.html`, `style.css`, `script.js`
- Also drag your `Divya_Mishra_CV.docx` file
- Click "Commit changes"

### Step 3 — Enable GitHub Pages
- Go to repository Settings
- Click "Pages" in the left sidebar
- Under "Source" → Select "Deploy from branch"
- Branch: `main`, Folder: `/ (root)`
- Click Save

### Step 4 — Your site is live!
- Wait 2-3 minutes
- Visit: `https://divas-coder.github.io`

## Customization
- Update email, LinkedIn, ResearchGate links in `index.html`
- Add your photo by placing `photo.jpg` and adding `<img src="photo.jpg">` in hero section
- Update publication DOIs as papers are published
- Change accent color in `style.css` by editing `--accent` variable
